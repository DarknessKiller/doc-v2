**Method :** <span style={{ color: "orange", fontWeight: "bold" }}>POST</span><br/>
Url : `https://sb-open.revenuemonster.my/v3/payment/reverse`

:::note
If transaction is timed out, should perform reverse order before creating new transaction. This is to make sure no double charge.
:::
